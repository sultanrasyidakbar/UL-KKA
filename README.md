# To-Do List (CLI)

## Deskripsi

Aplikasi CLI To-Do List untuk menambah tugas, melihat tugas, menandai tugas selesai, dan menghapus tugas.

## Fitur

- Tambah tugas
- Lihat semua tugas
- Tandai tugas selesai
- Hapus tugas
- Keluar

## Cara Menjalankan

1. Pastikan kamu punya Python ter-install.
2. Buka folder project (lokasi file `UL.py`).
3. Jalankan program:

```bash
python UL.py
```

## Cara Pakai (Menu)

Setelah program dijalankan, akan muncul menu:

- **1**: Tambah Tugas
- **2**: Lihat Semua Tugas
- **3**: Tandai Tugas Selesai
- **4**: Hapus Tugas
- **5**: Keluar

## Catatan

Setiap tugas disimpan dalam list di dalam program (belum tersimpan ke file).
